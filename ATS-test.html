About This App

This AI-powered system helps users evaluate how suitable their resume is for a given job.

  • Built with: Flask, Python, HTML/CSS/JS
  • Uses NLP and docx2txt for text extraction
  • Created by Anand Patel, 2025

How to Use

  • Upload your resume (.pdf or .docx)
  • Optionally upload the job description
  • Click “Analyze” to get your ATS score
  • If JD is skipped, only resume is scored

Note: Supported formats are .pdf and .docx only
